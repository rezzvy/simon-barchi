# SimonBarchi

Web-based Simon game with a touch of Blue Archive.

![Preview](https://repository-images.githubusercontent.com/725541645/fc91cacd-cc6e-490f-b731-60077f633a0e)

## Installation Steps

- Clone this repository.
- Unzip the downloaded files.
- Run `index.html` file.

Alternatively, you can play using our provided demo by clicking [this link](https://rezzvy.github.io/simon-barchi/).

## How to Play

- Click the **Start** button.
- Watch and wait until the game highlights each board; then, follow the sequence.
- Repeat the highlighted sequence.
- Continue playing until you make a mistake, and the game ends.
